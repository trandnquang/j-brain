# J-Brain: AI-Powered Japanese SRS 🧠🇯🇵

## 1. Mục tiêu
Dự án được xây dựng bởi **trandnquang**. Mục tiêu là rèn luyện kỹ năng **Fullstack (React/Spring Boot)** và ứng dụng **Local AI** để hỗ trợ lộ trình trở thành **BrSE** tại Nhật.

## 2. Tính năng MVP
- Tự động sinh Flashcard với 3 ngữ cảnh (Keigo, Daily, Anime) bằng Local AI.
- Thuật toán lặp lại ngắt quãng (SM-2) để tối ưu việc nhớ từ vựng N3.
- Giao diện tối giản (Global Basic style).

## 3. Cách chạy dự án
1. Mở LM Studio và Load model (Llama 3/Qwen).
2. Chạy lệnh: `make up`
3. Truy cập: `localhost:3000` (Frontend) và `localhost:8080` (Backend).
